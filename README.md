## 📖 О проекте

**JutliBot** — это первый шаг в разработке Telegram-ботов.  
Основная цель — научиться:
- работать с библиотекой `telebot` (PyTelegramBotAPI);
- подключать и обрабатывать данные от **Visual Crossing Weather API**;
- создавать удобные **Reply-клавиатуры** (кнопки управления);
- обрабатывать как команды, так и текстовые сообщения пользователей.

> ❗ **Важно:** Бот создавался исключительно в учебных целях. Токены в коде приведены для примера и в реальном проекте должны быть скрыты через `.gitignore`.

## 🚀 Что умеет бот?

### Команды и кнопки

| Команда / Кнопка | Действие |
|------------------|----------|
| `/start` | Приветствие и показ главного меню |
| `Старт` | Переход к выбору действий (Погода / Шутка) |
| `Информация` | Краткое описание возможностей бота |
| `Help` | Список доступных команд |
| `Погода` | Текущая погода в Челябинске (температура + описание) |
| `Шутка` | Случайная шутка из коллекции (про программистов 😄) |

### Пример работы

1. Пользователь нажимает `/start` → бот приветствует и показывает кнопки: `Старт`, `Информация`, `Help`
2. Пользователь нажимает `Старт` → появляются кнопки: `Погода`, `Шутка`
3. Пользователь нажимает `Погода` → бот запрашивает данные через **Visual Crossing API** и присылает:
   > 🌤 Текущая погода в Челябинске: -5°C, Partly cloudy
4. Пользователь нажимает `Шутка` → бот присылает случайную IT-шутку

## 🛠 Стек технологий

| Технология | Назначение |
|------------|------------|
| **Python 3.x** | Основной язык разработки |
| **pyTelegramBotAPI (telebot)** | Библиотека для работы с Telegram Bot API |
| **requests** | HTTP-запросы к погодному API |
| **Visual Crossing Weather API** | Получение данных о погоде (бесплатный тариф) |
| **ReplyKeyboardMarkup** | Удобные кнопки управления вместо текстовых команд |
